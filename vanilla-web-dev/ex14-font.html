font-family

字体族名

font-family: serif;
font-family: monospace;

font-size

大小

font-size: 1rem;
font-size: 2rem;

font-weight

粗细 normal/bold

font-weight: normal;
font-weight: bold;

font-style

样式 normal/italic

font-style: normal;
font-style: italic;

line-height

line 的高度略高于字体的高度是有必要的。因为在多行显示时，需要上下有多余空间才美观。

line-height: 1.0;
efg
123
123
123
line-height: 2.0;
efg
123
123
123

如果不指定 `vertical-align: top`，会出现 box 不对齐的情况。

line-height: 1.0;
efg
line-height: 2.0;
efg

letter-spacing

字符间的空间。一般不会改它。值越大，字符间隔越远；为负值表凑近。

letter-spacing: normal;
line-height: 0.2em;
